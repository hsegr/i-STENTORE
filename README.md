# i-STENTORE Middleware Repository

This repository contains the main middleware components developed by HSE for the i-STENTORE project.

## Components

- **iConnector**: A component for connecting various systems within the i-STENTORE ecosystem.
- **iBroker**: A messaging broker facilitating communication between different parts of the middleware.
- **iOntology**: Contains ontology definitions and related files, such as the i-STENTORE-Ontology.ttl, which define the semantic structure of the project.

## Structure

- `iOntology/`: Directory containing ontology-related files and definitions.
