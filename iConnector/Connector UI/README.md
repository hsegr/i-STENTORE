# Connector UI

The Connector UI provides an easy-to-use web interface for interacting with the i-STENTORE Data Space Connector.

Its purpose is to simplify connector management and common data-sharing operations without requiring direct interaction with the connector APIs or command-line tools.

Through the interface, users can perform common connector operations such as:

- browsing available assets
- managing data offerings
- reviewing contract negotiations
- monitoring transfer processes
- inspecting connector status

## Optional Component

The Connector UI is **optional**.

All connector functionality remains available through the underlying APIs provided by the Eclipse Dataspace Components (EDC) framework. The UI is intended to improve usability and simplify everyday operation for users who prefer a graphical interface over direct API or terminal interaction.
