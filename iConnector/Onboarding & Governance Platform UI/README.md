# Onboarding & Governance Platform UI

The Onboarding & Governance Platform provides the user interface for participant onboarding and governance within the i-STENTORE Data Space.

It supports both **Data Space participants** and **trusted issuers/administrators**, providing the workflows required to establish digital identities, issue credentials and manage participant admission throughout the lifecycle of the Data Space.

## Participant Portal

The participant interface provides a guided onboarding process, allowing organisations to:

- register and authenticate
- provide organisation information
- configure identity material
- generate and manage Decentralised Identifiers (DIDs)
- select trusted attestators
- review and submit onboarding requests
- monitor onboarding progress
- review issued Verifiable Credentials
- inspect onboarding status and related audit information

## Governance Portal

The administrator interface supports governance and operational workflows, including:

- reviewing participant admission requests
- approving or rejecting onboarding requests
- managing completed onboarding cases
- issuing Verifiable Credentials
- revoking previously issued credentials
- monitoring governance activities

## Purpose

The platform is more than a registration interface. It provides the governance workflows required to admit organisations into the i-STENTORE Data Space, establish their digital identities and manage the lifecycle of participant credentials after onboarding.

The credentials and identity material generated during onboarding are subsequently used by the i-STENTORE Connector Deployment to enable secure participation in the Data Space.
